# MMCP13
code description

This repository contains the code used for the experiments and results
presented in the paper:

**"Light scattering by random convex polyhedron in geometric optics approximation"**

A preprint of the paper is available at:  
https://arxiv.org/abs/2510.21201

**Author:** Quan Mu  
**Email:** mu.quan[at]foxmail[dot]com


## Code availability and usage

The code is made available for **academic research and educational use**
to support the reproducibility of the results presented in the paper.

**Commercial use is not permitted** without prior written permission
from the author.


## Intended scope

This code is intended for single-particle light scattering calculations
within the validity range of the geometric optics approximation.
It is primarily designed for academic research purposes and is not
optimized for large-scale production simulations or real-time applications.
